What is the difference between var and let?

The var variables are append to the windows or a function and we could redeclare and redefine these variables, and let variables are append to block-scopes and we could redefine the value but not redeclare the variable.

What is the difference between var and const?

The main difference is that var could be redeclared and redefine and these are append to the windows or a function and const could not be redefine or redeclared, and these are append to a block-scopes

What is the difference between let and const?

That with let variables we could redefine the values of the variables and with const not.

What is hoisting?

It’s the way that we explain how the JS compiler works. Variables are lifting or “hoisted” to the top of the scope they are declared in. When using var, you can access the variable name and it’s undefined value before it is used later on.
